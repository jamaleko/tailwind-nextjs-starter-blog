<h2>Introduction</h2>
<p>Kembang api memang memiliki daya tarik yang luar biasa, namun setelah ledakan terakhir sirna dan kerumunan bubar, kembang api ternyata meninggalkan sesuatu yang tidak terlihat namun bertahan jauh lebih lama dari pesta itu sendiri, yaitu polusi di udara dan air.</p><h2>Dampak Kembang Api pada Lingkungan</h2>
<p>Beberapa penelitian yang baru saja diterbitkan di jurnal-jurnal ACS mengungkap apa yang sebenarnya tersisa setelah kembang api, mulai dari sampah dan partikel di udara hingga bahan kimia yang dilepaskan ke atmosfer. Bersama-sama, temuan-temuan ini melukiskan gambaran yang lebih jelas tentang bagaimana kembang api bisa memengaruhi lingkungan dan kesehatan manusia.</p><h2>Pengaruh Kembang Api pada Kualitas Air</h2>
<p>Penelitian laboratorium yang diterbitkan dalam Environmental Science & Technology menemukan bahwa residu petasan ini bisa mengubah kimia air secara signifikan jika masuk ke danau atau sungai. Para peneliti menemukan bahwa residu petasan melepaskan ion logam seperti kalium dan mangan, serta materi organik terlarut seperti fenol sederhana dan senyawa yang mengandung belerang ke dalam air.</p><h2>Dampak Kembang Api pada Kualitas Udara</h2>
<p>Para peneliti yang melaporkan temuan mereka di ACS ES&T Air melakukan pemantauan partikel udara selama acara atletik besar multi-hari di Inggris. Mereka mendeteksi peningkatan jangka pendek yang cukup mencolok pada partikel udara kasar maupun halus. Sebagian besar polusi berasal dari emisi masak para pedagang makanan dan debu yang berterbangan akibat kendaraan. Tapi selama upacara pembukaan dan penutupan, muncul dua lonjakan partikel halus yang jelas.</p><h2>Kontribusi Kembang Api pada Kabut Asap Atmosfer</h2>
<p>Beberapa formulasi kembang api mengandung senyawa yang disebut amina. Senyawa ini bisa bereaksi di atmosfer untuk membentuk aerosol yang memperburuk kualitas udara. Untuk mencari tahu apakah senyawa ini hancur saat ledakan atau justru terlepas ke udara, para ilmuwan mengukur gas dan partikel udara selama perayaan Tahun Baru Imlek di kawasan pinggiran China. Temuan mereka menunjukkan lonjakan tajam pada beberapa amina dibandingkan dengan periode tanpa perayaan, terutama selama pertunjukan kembang api terbesar.</p><h2>Penutup</h2>
<p>Hasil ini menunjukkan bahwa kembang api berkontribusi lebih dari sekadar asap yang terlihat pada kabut yang bisa bertahan lama setelah perayaan berakhir. Oleh karena itu, perlu dilakukan pengumpulan dan pembuangan sisa petasan yang tepat untuk mengurangi dampak lingkungan ini. Selain itu, perlu dilakukan penelitian lebih lanjut untuk memahami dampak kembang api pada lingkungan dan kesehatan manusia.</p>
