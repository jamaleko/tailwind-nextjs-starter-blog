<h2>Penelitian Ilmuwan</h2>
<p>Rumah api di Sleman menjadi fenomena yang menarik perhatian banyak orang. Pusat Kajian Pelambaran Entropi (PKPE) FT UGM telah melakukan penelitian untuk mengungkap penyebab kemunculan api di rumah warga di Seyegan, Sleman. Hasil penelitian menunjukkan bahwa api tidak berkaitan dengan gas, melainkan dengan resin poly vinyl chloride (PVC).</p>
<h2>Penyebab Api</h2>
<p>Sebelumnya, tim PKPE FT UGM sempat menduga bahwa fenomena api di rumah Fia berhubungan dengan keberadaan gas hidrogen yang berasal dari limbah pemotongan ayam. Namun, skenario itu kini ditepis oleh Ketua tim PKPE FT UGM, Prof Alva Edy Tontowi. "Sumber api bukan dari rembesan gas alam dari bawah permukaan (lantai), tidak ada anomali termal dan tidak ditemukan gas yang dapat menyala sendiri secara alami (self-ignition) pada suhu kamar," jelasnya.</p>
<h2>Prinsip Teori Segitiga Api</h2>
<p>Mengacu pada Prinsip Teori Segitiga Api dan hasil penelitian, pihak PKPE FT UGM menyimpulkan bahwa medan elektromagnetik terukur pada level aman. Ini berarti bahwa bukan sumber pemantik nyala api. Kemungkinan besar, api diasosiasikan dengan adanya resin PVC yang mudah terbakar bila bertemu sumber api atau ignition. Sebab, ditemukan resin PVC pada material yang terbakar.</p>
<h2>Hasil Penelitian</h2>
<p>Prof Alva Edy Tontowi menjelaskan bahwa "resin ini telah ditemukan pada residu pembakaran berdasarkan pengujian metoda FTIR." Pihaknya juga telah menganalisis beberapa material terbakar menggunakan metode Headspace GC (Gas Chromatography). Hasilnya, material itu dipastikan tidak terbakar karena gas hidrogen. "Hasil analisis hanya dapat mendeteksi adanya gas CO2 dan tidak ditemukan unsur-unsur sisa hidrokarbon atau solven yang bisa dipakai sebagai akseleran kebakaran," tutur Alva.</p>
<h2>Analisis FTIR</h2>
<p>Lebih lanjut, hasil analisis FTIR menunjukkan bahwa sampel-sampel residu pada permukaan dinding keramik maupun kayu/tripleks menunjukkan kandungan PVC yang tidak umum dijumpai pada permukaan tersebut. Dengan begitu, hasil ini menjadi akhir dari penelitian fenomena api di rumah Muftia di Segeyan, Sleman. Hasilnya sudah diserahkan pula ke BPDB Sleman.</p>
<h2>Pengamatan Spasial</h2>
<p>Secara spasial dari hasil observasi ketinggian menggunakan wahana drone dan sensor Thermal Infrared yang dilakukan pada dinihari di lokasi kemunculan api hingga radius 200 meter disekitarnya, tidak menunjukkan adanya anomali termal. Ini menunjukkan bahwa api tidak memiliki sumber yang jelas dan tidak ada indikasi adanya kebocoran gas atau sumber panas lainnya.</p>
<h2>Kesimpulan</h2>
<p>Dengan demikian, misteri api di Sleman telah terpecahkan. Ilmuwan telah menemukan penyebabnya, yaitu resin PVC yang mudah terbakar. Hasil penelitian ini dapat menjadi acuan bagi masyarakat untuk meningkatkan kesadaran akan bahaya kebakaran dan pentingnya menjaga keamanan lingkungan.</p>
